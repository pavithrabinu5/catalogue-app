# Catalogue App

An educational animal information browser with swipeable cards and category-based filtering.

**Features:**
- ViewPager2 with RecyclerView.Adapter pattern
- Category filtering (All, Mammals, Aquatic) using Material ChipGroup
- Synchronized icon navigation with click-to-jump
- OnPageChangeCallback for state synchronization
- Displays scientific names, habitats, and fun facts

**Tech Stack:** Java, Android SDK, ViewPager2, Material Design Components
